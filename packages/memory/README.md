# Memory Package

This package will implement task-local and cross-task memory.

Memory classes:

- Short-term: current job plan, tool outputs, Agent context, repair history.
- Long-term: project-level patterns, fixes, preferences, and known build/runtime behavior.
- Entity: projects, domains, modules, components, dependencies, APIs, symbol families.
- Scenario: similar failures, browser runtime errors, sandbox diagnoses, repair cases.
